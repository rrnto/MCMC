# Metropolis-Hastings vs emcee for Linear Model Sampling

Here I tried to compare the Metropolis–Hastings algorithm that I implemented with `emcee` for sampling a linear model. The project demonstrates:

- Implementation of Metropolis–Hastings MCMC
- Use of `emcee` for ensemble sampling
- Comparison of trace plots, posterior histograms, and corner plots
- Parameter estimation with median and 16th/84th percentiles

## Requirements
- Python 3.x
- Jupyter Notebook
- numpy
- matplotlib
- scipy
- emcee
- corner
